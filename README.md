# 🤖 AI Chat System 

An advanced AI-powered chat system built using a **React frontend** and **Flask backend**, capable of delivering intelligent responses in both **online and offline modes**. The system is designed with scalability, performance, and real-world usability in mind.

---

## 🚀 Features

* 💬 Real-time chat interaction
* ⚛️ Modern React-based frontend UI
* 🤖 AI-powered intelligent responses
* 🌐 Online mode (API-based responses)
* 📴 Offline mode (local model / fallback system)
* ⚡ Fast and scalable Flask backend
* 🎨 Clean, responsive, and user-friendly interface
* 🔄 Seamless switching between online & offline modes

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash id="c6u1bl"
git clone https://github.com/dev-balamurugan06/ai-chat-system.git
cd ai-chat-system
```

---

## 🔧 Backend Setup (Flask)

```bash id="0bj1df"
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

👉 Backend runs on:
`http://localhost:5000`

---

## ⚛️ Frontend Setup (React)

```bash id="w41i8q"
cd frontend
npm install
npm start
```

👉 Frontend runs on:
`http://localhost:3000`

---

## 🌐 Usage

* Open browser → `http://localhost:3000`
* Start chatting with AI 🤖
* System automatically:

  * Uses **online AI** when internet is available 🌐
  * Switches to **offline mode** when no internet 📴

---

## 🧠 Hybrid AI Working

| Mode       | Description                                  |
| ---------- | -------------------------------------------- |
| 🌐 Online  | Uses external AI APIs for advanced responses |
| 📴 Offline | Uses local logic/model for basic responses   |
| 🔄 Hybrid  | Automatically switches based on availability |
